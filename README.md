# Plane Builder

## Getting Started

1. **Clone the Repository**

```bash
git clone https://github.com/katsugrace/PlaneBuilder.git
cd PlaneBuilder
```

2. **Build packages**

```bash
colcon build
source install/setup.bash
```

3. **Run launch file**

```bash
ros2 launch plane_builder_launch plane_builder.launch.py 
```
